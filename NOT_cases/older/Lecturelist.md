                                    <h1 id="page-title" class="asset-name entry-title">The Lecture List</h1>
                                </div>

                                <div class="asset-content">

                                    <div class="entry-body">
                                        <p>The Lecture List was launched in May 2004. Over the twelve-month design and development phase, Dug was responsible for the entire user experience, from functionality specification through to application design, human interaction and control of the final coding of the site's interface.</p>

<p><img src="http://donkeyontheedge.com/images/book/lect364_1.gif" alt="The Lecture List website start page" height="273" width="364" /></p>
                                    </div>


                                    <div id="more" class="entry-more">
                                        <h2>Special features</h2>

<p>The system is designed to prove that an accessible site can be as beautiful and engaging as it is useful. The layout is all <span class="caps"><span class="caps">CSS.</span></span> It uses no html tables for non-tabular data and deploys separate stylesheets for different uses. All parts of the design either shuffle or compress to allow for acceptable viewing on a mobile phone.</p>

<p><img src="http://donkeyontheedge.com/images/book/lect180_1.gif" alt="" height="135" width="180" /><img src="http://donkeyontheedge.com/images/book/lect180_2.gif" alt="" height="135" width="180" /></p>

<p>The information architecture was challenging by definition in that the database was&#8212;in theory&#8212;going to contain the sum of all knowledge. The final design is a tree structure with a network of primary &#8216;roots' that each grow branches as users create new subtopics. The extensive and potentially quite &#8216;dry' data is presented in a magazine format which allows posters to use photographs and html typography.</p>

<p><img src="http://donkeyontheedge.com/images/book/lect180_5.gif" alt="Greg Dyke" height="135" width="180" /><img src="http://donkeyontheedge.com/images/book/lect180_4.gif" alt="The path to marine mammals is exposed by the browsing interface" height="135" width="180" /></p>

<p>Organisers post events which then proceed through a moderation queue.</p>

<p><img src="http://donkeyontheedge.com/images/book/lect364_5.gif" alt="Add event" height="36" width="360" />
<img src="http://donkeyontheedge.com/images/book/lect364_6.gif" alt="Review posted events" height="36" width="360" />
<img src="http://donkeyontheedge.com/images/book/lect364_2.gif" alt="Edit and correct interface" height="36" width="360" /></p>

<p>The site comes with a notification system that visitors activate by viewing events and then subscribing to them.</p>

<p><img src="http://donkeyontheedge.com/images/book/lect364_3.gif" alt="Browse events" height="36" width="360" />
<img src="http://donkeyontheedge.com/images/book/lect364_4.gif" alt="Manage subscriptions" height="36" width="360" /></p>

<p>Any questions about the system may be addressed to <a href="mailto:%62%69%6C%6C%79%40%6C%65%63%74%75%72%65%6C%69%73%74%2E%6F%72%67">Billy Clark</a> </p>
                                    </div>
