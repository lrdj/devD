                                    <h1 id="page-title" class="asset-name entry-title">Twocultureclash music store interface</h1>
                                </div>

                                <div class="asset-content">

                                    <div class="entry-body">
                                        <p><img src="http://donkeyontheedge.com/images/book/2cult180_1.gif" alt="twocultureclash.com" class="left" style="float:left;border:none;width:170px!important;" /></p>

<p>With the Twocultureclash shop, Dug worked closely with YR Media's Mark Panay to really push the white-label application.</p>

<p>Mark's semantic markup allowed Dug to build a design around <span class="caps">CSS </span>selectors acting on most parts of the site without need for specific class definitions.</p>

<p>Dug used <span class="caps"><span class="caps">CSS </span></span>attributes to create a rich design which surprised both the record label and the client. This design solution has set the standard and Dug is now developing further online ringtone shops with <span class="caps"><span class="caps">YRM</span></span>edia.</p>

<p><img src="http://donkeyontheedge.com/images/book/2cult364_1.gif" alt="twocultureclash.com website" /></p>

<p>The microsite is linked to from <a href="http://www.twocultureclash.com/">twocultureclash.com</a> website and via a <a href="javascript:window.open('http://www.yrmedia.com/theshop/shop.asp?sid=2f5a110002b4e72e3ffe3a5aeae72e74','','width=800,height=600,status=1')">pop-up link</a>.</p>

<p>See also <a href="/portfolio/recent/all_css_mia_ringtone_store_interface.html">miauk.com project</a></p>
